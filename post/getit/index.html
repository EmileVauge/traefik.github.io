Get it

Fri, Sep 11, 2015
  • The simple way: go to the Github releases page and get a binary for your platform. Just run it!
./traefik traefik.toml
  • Use the tiny Docker image:
docker run -t -p 8080:8080 -p 80:80 -v $PWD/traefik.toml:/traefik.toml emilevauge/traefik
  • From sources:
git clone https://github.com/EmileVauge/traefik
